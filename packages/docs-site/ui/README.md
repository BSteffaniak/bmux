# bmux_docs_site_ui

UI components and templates for the bmux documentation website.

## Overview

Presentation layer for the docs site, organized into layout templates, navigation
components, page content modules, and theme definitions. Used by the
`bmux_docs_site` crate to render each documentation page.

## Modules

- **`layout`**: Page shell, header, footer, and content wrappers
- **`nav`**: Sidebar navigation and breadcrumb components
- **`pages`**: Content for each documentation route
- **`theme`**: Color scheme and typography definitions
